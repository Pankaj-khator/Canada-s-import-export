# Canada Import-Export Analysis Dashboard

**Power BI Dashboard Link:** [View Dashboard](https://app.powerbi.com/groups/me/reports/2d4c912c-1617-4142-93a3-5b2e277a5021?pbi_source=desktop)

---

## Objective

The primary objective of this project is to develop a user-friendly interface for exploring Canada’s import and export activities with over **220 trading partners** and **5,500 products**. The dashboard provides stakeholders with actionable insights into Canada’s international trade dynamics, enabling informed decision-making. The project also highlights the **top three products** and **top three trading partners** in Canada's trade portfolio.

---

## Project Description

This project provides a comprehensive analysis of Canada’s import and export trends from **2002 to 2022**, uncovering the dynamics shaping the nation’s international trade landscape.  

Working with datasets exceeding **100 million records**, the project faced challenges in **data storage, cleaning, modeling, and visualization**. Overcoming these challenges was critical to extracting meaningful insights.

Data was sourced from **Statistics Canada**, leveraging **Power BI models** integrated with **Harmonized System six-digit (HS6) codes**. This approach allows users to explore product categories in detail, providing a granular understanding of Canada’s import and export activities across various sectors.  

The dashboard combines **advanced data modeling** with **interactive visualizations**, enabling users to identify trends, patterns, and anomalies, as well as understand the economic factors driving trade over the past two decades.

## Key Features

- **Comprehensive Data Analysis:** Explore Canada’s import and export flows, trends, and patterns over time.  
- **Customizable Dashboard:** Users can tailor the dashboard to focus on specific trade data and metrics.  
- **Paid Customization Options:** Advanced features and deeper insights are available through optional paid services.  
- **Correlation Analysis:** Conduct correlation analysis between two products or countries to identify trade relationships and dependencies.
